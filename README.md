This Daily Quote Generator offers a refreshing and stylish way to get your daily dose of inspiration. It includes:

“New Quote” Button: A button that instantly generates a fresh, motivational quote along with the author’s name.

The button is accompanied by a sync icon (🔄), making it intuitive and visually appealing.
“Tweet” Button: Once you’ve found the quote you love, share it with the world by clicking the tweet button. The button features a Twitter icon (🐦) for easy access to tweet your quote directly on Twitter.

Quote and Author Display:

The quote is elegantly wrapped in quotation marks (❝ ❞) and presented in a stylish italic font, giving it a classic, literary feel.
The author’s name is displayed below the quote in a subtle, right-aligned text to signify attribution.
Design Elements:

Clean, minimalist design with a soft background color to keep focus on the quote.
Buttons are designed with vibrant colors, including a green New Quote button (🔄) and a blue Tweet button (🐦), enhancing the aesthetic while making the interface user-friendly.
Hover effects and smooth transitions for the buttons, giving it a modern and interactive look.
